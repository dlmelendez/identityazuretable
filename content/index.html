Project Description

This project provides a high performance cloud solution for ASP.NET Identity Core using Azure Table storage replacing the Entity Framework/MSSQL provider

ElCamino.AspNetCore.Identity.AzureTable (ASP.NET Identity Core framework)
Use this package to target > = netstandard2.0, net5.0 and Microsoft.AspNetCore.Identity for AspNetCore.Mvc

ElCamino.AspNet.Identity.AzureTable (ASP.NET Identity 2.0 framework)
(deprecated) Use this package to target the .NET Framework 4.5 and Microsoft.AspNet.Identity.Core 2.2 for Mvc 5.

This project is a plug-in to the ASP.NET Identity framework. A basic understanding. of the ASP.NET identity framework is required.

By default, the ASP.NET Identity system stores all the user information in a Microsoft SQL database using an EntityFramework provider. This project is a replacement of the EntityFramework SQL provider to use Azure Table Storage to persist user information such as (but not limited to): username/password, roles, claims and external login information.

In summary, the ASP.NET Identity Core Azure Table Storage provider allows one to remove the EntityFramework and Microsoft SQL dependencies from the default implementation of the ASP.NET Identity Core storage provider and replace it with a high performance Azure Table Storage solution.
